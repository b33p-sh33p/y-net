Make tabs look like You
Note: Doesn't work with "chrome://" websites
Step 1: Copy the code below

javascript:(function() { var link = document.querySelector("link[rel*='icon']") || document.createElement('link'); link.type = 'image/x-icon'; link.rel = 'shortcut icon'; link.href = 'https://s.ytimg.com/yts/img/favicon_96-vflW9Ec0w.png'; document.title='YouTube'; console.log(document.title); document.getElementsByTagName('head')[0].appendChild(link); })();

Step 2: Right-click (laptop users: click with 2 fingers) the bookmarks bar, and select "Add page"

Step 3: Choose the bookmark title, then in the "URL" part replace the existing text with the code you copied

There! If you ever need to disguise a tab as YouTube , click the bookmark!
